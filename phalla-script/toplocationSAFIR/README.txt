
This script will sort data ascending from csv file to get the top location

I- REQUIREMENTS

	Install pandas and python2.7

II- DESCRIPTION

	- TopUserNames.py: will sort data to get top location and write it to csv file.

III- INPUT

	- safirstores_follower.csv: list of data that you want to sort.

IV-OUTPUT

	Three csv file with different top location.

V- HOW TO RUN IT

	1. Open terminal and go to your directory.
	2. Launch the script python writecsvfrompandas.py
