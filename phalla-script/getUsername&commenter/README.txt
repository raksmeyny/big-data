
This script will get username and commenter from instagram profile.

I- REQUIREMENTS

Install selenium and python2.7

II- DESCRIPTION
	
	- link_comment.py: this script will collect all link from each username profile posted
	- comment.py: this script will collect all commenter and username
	- run.sh: this script will call link_comment.py and comment.py.

III- INPUT

	- username.csv: list of user profile we need to track. (We track their commenters)

IV-OUTPUT

A csv file store all link comment and a json file store all commenter and their comment.

V- HOW TO RUN IT

1. Open terminal and go to your directory.
2. Launch the script ./run.sh
