This script is matching safir products with beautypedia.

I- REQUIREMENTS
	
	Install selenium and python2.7

II- DESCRIPTION

	-getPau.py: get all the products to search with google for matching with beautypedia.
	-get_review.py: to get the product details and product reivew.
	-newTitle.csv: list all safir product.
	-launch.sh: run the script
	

III- INPUT

	-newTitle.csv: store list of safir products.


IV-OUTPUT

	-link.csv: all link that get from getPau.py.
	-product.json: store all products detail.
	- one more is product review it will creat file json follow the brand name (EX: rojashop.json)
	
V- HOW TO RUN IT

1. Open terminal and go to your directory.
2. Launch the script ./launch.sh
