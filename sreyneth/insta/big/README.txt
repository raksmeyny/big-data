This script will get user profile, post details and user profile of all user commented in annanemati. 

I- REQUIREMENTS
	
	Install selenium and python2.7

II- DESCRIPTION

	- user_profile.py: get user profile of annanemati.
	- user_profile_detail.py: get post detail of annanemati.
	- ucomment.py: get all user profile of all username user comment in annanemati posted.
	- launch.sh: lauch all the script above with key name (annanemati).

III- INPUT

IV-OUTPUT

	- userprofile.json: store user profile data of annanemati.
	- post_detail.json: store post details data.
	- user_comment.json: store user profile of username user comment in annanemati.
	
V- HOW TO RUN IT

1. Open terminal and go to your directory.
2. Launch the script ./launch.sh
