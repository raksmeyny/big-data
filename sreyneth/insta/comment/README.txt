This script will get 100 posts and in each post it will get all the comment of rojashop.

I- REQUIREMENTS
	
	Install selenium and python2.7

II- DESCRIPTION

	- link_comment.py: it scroll page to get all link of 100 posts.
	- comment.py: get all the comment in each post.
	- launch_com.sh: lauch all the script above.

III- INPUT

IV-OUTPUT

	- comment.json: stores all the link and all the comment of rojashop.
