This script will get all user profile data of username in safir. 

I- REQUIREMENTS
	
II- DESCRIPTION

	- user_profile.py: get user profile data (username, average of like, website).
	- user_profile_detail.py: get post detail data.
	- safir.csv: list all username in safir.
	- launch.sh: lauch all the script above with key name.

III- INPUT

	- safir.csv: store all list username in safir.

IV-OUTPUT

	- user_profile.json: store user profile data.
	- postl.json: store post details data.

V- HOW TO RUN IT

1. Open terminal and go to your directory.
2. Launch the script ./launch.sh
